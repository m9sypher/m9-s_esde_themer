#Ver (alpha)0.4

M9 THEME EDITOR — CHANGELOG

────────────────────────────

- Added Live Preview — a new button that actually simulates ES-DE running your currently loaded theme. Opens a separate window with a System View and Gamelist View, navigable with on-screen D-pad and A/B/X/Y controls
  - Left/Right in System View cycles through your real systems (from your loaded ROMs/gamelists); A enters that system's gamelist, B backs out
  - Up/Down in Gamelist View cycles through that system's real games, live-updating any metadata-bound text/image/rating elements
  - If multiple gamelist or system view files are open, dropdown selectors let you switch between them mid-preview — switching triggers a "Reloading theme..." moment first, same as switching a variant does in real ES-DE
  - Shows a clear error instead of a blank window if no theme content is loaded at all
- Fixed a real bug: Live Preview wasn't applying the theme's actual fonts — now resolves and applies real fontPath fonts the same way the main canvas does
- Fixed a real bug: carousel/grid in Live Preview only ever showed game names — now resolves and shows real game art per the element's imageType, falling back to a name label only when no art is available
- Fixed a real bug: the gamelist textlist in Live Preview didn't follow the highlighted game — now auto-scrolls to keep the current selection visible
- Fixed text vertical-centering to match ES-DE's real behavior (confirmed against ES-DE's own source): a text box only gets centered when it's shorter than one line actually needs — a genuinely multi-line-capable box stays top-anchored as normal

────────────────────────────

KNOWN TIPS / GOTCHAS

- Live Preview's file selectors only list files that are actually open in the editor — having a file on disk isn't the same as having it open. Open it in the Files panel first if it's not showing up as an option.
- size stretches to exact dimensions with no aspect preservation; maxSize fits within bounds while keeping the real aspect ratio; cropSize (+ optional cropPos) scales up and crops to fill — pick whichever matches what you actually want.
- Adding a new variant through Capabilities on an existing theme auto-wires it into theme.xml on Save — that patch only happens on Save, not automatically as you type.
- View names are only ever "gamelist", "system", or "all" — different looks are separate variant files, not view names.
