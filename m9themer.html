Find & replace across theme ✕

Plain text match, not regex. Works across every gamelist file in the theme — useful for renaming a ${token} everywhere at once. A backup .zip of every affected file downloads automatically before anything is changed.

 Variables ✕

The shared colors and font paths your theme's elements reference as ${tokens}, like ${colorText} or ${regularFont}. If your theme defines multiple color schemes (e.g. Dark/Light), each one gets its own set of these — pick which to edit below.

 Capabilities ✕

capabilities.xml is mandatory — ES-DE won't load a theme without it. Everything declared here can then be referenced by name inside your gamelist/theme XML.

 Log Checker ✕

Pick a real ES-DE log file (or paste its contents below) and this does a best-effort text search for lines that look like a theme-related error. This isn't a verified ES-DE log parser — just a plain keyword scan — so treat it as a starting point, not a definitive diagnosis.

 Resolution Declarations ✕

A decent built-in list of real resolutions to preview against, plus your own custom ones — helps check pixel-level accuracy against an actual target device instead of just an aspect ratio.

 Sounds ✕

Standard ES-DE navigation sounds — scroll, select, back, etc. Set a file for each slot you want, then Save writes (or creates) navigationsounds.xml in your theme folder automatically.
