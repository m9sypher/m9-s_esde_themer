Find & replace across theme ✕

Plain text match, not regex. Works across every gamelist file in the theme — useful for renaming a ${token} everywhere at once. A backup .zip of every affected file downloads automatically before anything is changed.

 Variables ✕

The shared colors and font paths your theme's elements reference as ${tokens}, like ${colorText} or ${regularFont}. If your theme defines multiple color schemes (e.g. Dark/Light), each one gets its own set of these — pick which to edit below.

 Capabilities ✕

capabilities.xml is mandatory — ES-DE won't load a theme without it. Everything declared here can then be referenced by name inside your gamelist/theme XML.

 Asset browser ✕

Every image found in your theme folder. Click one to drop it onto the canvas as a new image element pointing at that file.

 Snippets ✕

Reusable saved elements. Right-click any element on the canvas → "Save as snippet" to add one here. Click a snippet to drop a copy onto the canvas.
